## 代码由来
这是我手动配置一个webpack应用的小实例，它具备基础的热加载、路由功能

起初写这个小实例用来体验webpack的配置，尝试搭建一个简单的模块化加载的小框架；

后来也用来记录一些对知识点的学习、尝试等，比如ES6的学习与小实例尝试；

## Build Setup
``` bash
# Project setup
npm install

# Compiles and hot-reloads for development
npm run serve

# Compiles and minifies for production
npm run build
```
